1. Estrutura Básica do HTML

voltar

A estrutura básica de um documento HTML é composta por elementos que definem a estrutura e o conteúdo de uma página web. É essencialmente composta por três partes principais: a declaração do tipo de documento, a tag HTML e as tags head e body.

1.1 Declaração do Tipo de Documento (DOCTYPE):

voltar

A declaração informa ao navegador que o documento é um HTML5. Esta declaração deve ser a primeira linha do documento HTML.

1.2 Tag HTML:

voltar

A tag <html> engloba todo o conteúdo da página HTML. É onde se define o idioma da página (através do atributo lang).

1.3. Tag Head:

voltar

A tag <head> contém informações sobre o documento que não são visíveis ao usuário, como: Título da página (<title>). Metadados (<meta>). Links para folhas de estilo (CSS) e scripts (<link> e <script>).

1.4. Tag Body:

voltar

A tag <body> contém o conteúdo visível ao usuário, como: Texto. Imagens. Links. Formulários. Outros elementos que compõem a página.

2. Tags HTML

voltar

As tags HTML são como "etiquetas" que delimitam os elementos de uma página web. Elas indicam ao navegador como estruturar o conteúdo, como títulos, parágrafos, imagens, links e muito mais.

Estrutura básica: As tags são escritas entre colchetes angulares <>. Algumas tags precisam de fechamento, enquanto outras são auto-fechadas.

3. Atributos das tags HTML

voltar

Os atributos nas tags HTML são informações adicionais que você inclui na tag de abertura para controlar o comportamento, as características e a funcionalidade do elemento. Eles fornecem detalhes específicos sobre como o elemento deve ser interpretado ou exibido.
